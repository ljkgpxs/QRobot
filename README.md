###安装
安装依赖<br />
##Ubuntu:
```
$ sudo apt-get install libpng12-dev libcurl4-openssl-dev
```

##Centos:
```
# yum install libpng12-devel libcurl-devel
```
安装Mojo-Webqq依赖，项目地址: [Mojo-Webqq](https://github.com/sjdy521/Mojo-Webqq)
<br />

编译：<br />
需要gcc编译器
```
$ ./build.sh
```

`perl版本最低要求 5.10`
###使用
```
$ ./start.pl
```
根据提示进行账号键入和二维码扫描
